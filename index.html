Memo é uma função utilitária do framework React que utiliza o conceito de memorização para evitar calculos computacionais repetidos, cacheando o resultado e o retornando.

Quando "Memo Change Color" recebe os mesmos valores de agumento não executará uma nova conversão de RGB para Hex, irá retornar diretamente o resultado em cache.

Importante destacar deve ser composto de funções puras, os mesmos valores de entrada devem sempre gerar a mesma saída.
